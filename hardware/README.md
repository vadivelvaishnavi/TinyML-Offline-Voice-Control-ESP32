# Hardware Design

This folder contains the hardware setup and circuit design files for the TinyML voice controller system.

## Files

### `circuit_diagram.png`
Complete ESP32 circuit diagram showing:
- INMP441 microphone connections
- OLED display wiring
- LED/Relay connections
- Power supply configuration

### `hardware.png`
Physical hardware prototype setup.

## Main Hardware Components

- ESP32 Dev Board
- INMP441 I2S Microphone
- SSD1306 OLED Display
- Relay Module
- LED Indicator
- Breadboard & Jumper Wires

## Hardware Features

- Real-time audio capture
- Low-power operation
- Edge AI processing
- Embedded TinyML inference
- Offline functionality
